COinS

COinS (Content Object in Span) is a way of putting some metadata inside a plain HTML document; essentially it is a specially crafted, empty span element. As the specification makes clear, the span element was chosen to carry this data because it is “invisible to the user and innocuous with respect to HTML markup.” All the metadata is tucked into the “title” attribute on a span; and the metadata itself is formatted according to the NISO 1.0 Context Object standard.
